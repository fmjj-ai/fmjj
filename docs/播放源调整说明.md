# 曲目默认播放源调整说明

## 背景

原先曲目区默认播放源由 `assets/js/app.js` 的 `firstPlayableIndex` 决定，它取
`players` 数组里第一个可嵌入项。多数曲目的第一个是 Spotify，所以默认播 Spotify。

需求：默认优先 Bilibili，其次 Apple Music，其余平台（Spotify / YouTube / 网易云搜索）
全部保留为可切换标签，不删除。本地其他版本能补的先补，本地没有的再联网搜索。

## 已完成的本地改动

### 1. 默认播放源优先级（一处改动，全局生效）

文件：`assets/js/app.js`（`firstPlayableIndex` 与新增的 `playerRank`，约 213-241 行）

- 新增优先级 `PLAYER_PRIORITY = ['bilibili', 'apple']`。
- 选默认源时按 Bilibili > Apple Music > 其他可嵌入源 排序，命中最高优先级的播放器。
- 都没有 B 站 / Apple 时，退回原逻辑（第一个可嵌入项）。
- Spotify、YouTube 等仍作为标签保留，用户可手动切换。
