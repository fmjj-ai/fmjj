# 页面设计提示词

为「封茗囧菌 / Mandi Sa」制作一个个人音乐网页。整体以淡黄色、淡粉色、薄荷青和奶油白为主色，加入少量薰衣草紫作为文字和阴影。风格关键词：甜美、清透、国风音乐、二次元与真人资料并置、玻璃拟态、漂浮音符、柔和光晕、滚动揭示、卡片式布局。

必须包含：

1. Hero 首屏：姓名、英文名、身份标签、代表作、主视觉图片、漂浮音乐符号。
2. 个人档案：全网 ID、英文名、昵称、生日、星座、代表作、商务联系，并区分可信来源与二级资料。
3. 热门曲目：不同平台的热门曲目采样，不混成绝对排名。
4. 网页内播放器：点击曲目后切换 Apple Music / Spotify / Bilibili / YouTube iframe，不跳转。
5. 平台入口：微博、B站、5sing、网易云、QQ音乐、酷狗、酷我、Apple Music、Spotify、YouTube Music、抖音等。
6. 图片素材库：使用本地 assets/images 图片，卡片悬停放大，点击打开 lightbox。
7. 时间线：从早期翻唱、入驻B站、代表作、近年活动到最新发行。
8. 来源区：列出所有来源链接和可信度。

动效：优先使用 GSAP + ScrollTrigger；CDN 不可用时使用 CSS transition 和 IntersectionObserver 降级。鼠标光晕、漂浮音符、卡片 hover、滚动 reveal 都要有。
