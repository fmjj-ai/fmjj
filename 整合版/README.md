# 封茗囧菌 · Mandi Sa 整合版网页

打开方式：直接双击 `index.html` 即可查看。

本网页整合了四个原始项目的优点：
- 资料完整型：来源说明、完整数据、本地图库、站内 iframe 播放器；
- 现代动效型：玻璃拟态、移动端导航、渐入动画、设计系统；
- 视觉沉浸型：BGM、音乐分类、丰富相册、时间线叙事；
- 轻量核验型：主题切换、曲目搜索、隐私/核验策略、低依赖实现。

文件结构：
- `index.html`：主页面。
- `assets/css/style.css`：页面样式。
- `assets/js/data.js`：页面数据。
- `assets/js/app.js`：交互逻辑。
- `assets/images/`：本地图像素材。
- `audio/bgm.mp3`：背景音乐素材，来自用户上传的原网页。
- `docs/ANALYSIS_AND_ARCHITECTURE.md`：四站分析与整合架构说明。

注意：本页为粉丝向静态网页示例，不代表艺人官方站点。音乐播放使用第三方平台公开 iframe 或跳转，不下载或转存受版权保护的歌曲文件。
