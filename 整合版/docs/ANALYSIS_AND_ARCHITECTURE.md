# 四个封茗囧菌网页分析与整合架构

## 1. fengming_jiongjun_site_full
- 定位：资料完整型、数据驱动型。
- 优点：本地图库、完整 JSON、平台列表、音乐播放器 iframe、多来源说明、图片灯箱、部署文件齐全。
- 适合保留：数据结构、来源/版权说明、站内播放器、本地图片素材库。

## 2. fengming-jiongjun-site
- 定位：现代动效型、玻璃拟态型。
- 优点：跳转主内容、移动端导航、关键词跑马灯、卡片式资料、播放器源切换、设计系统说明、来源警告。
- 适合保留：导航体验、玻璃卡片、渐入动效、设计系统与版权提醒。

## 3. 封茗囧菌个人网页
- 定位：视觉沉浸型、内容丰富型。
- 优点：WebGL 背景、背景音乐、音乐分类 Tabs、弹窗播放器、音频可视化、DNA 时间线、16 张以上相册素材、平台卡片与商务信息。
- 适合保留：视觉氛围、音乐分类、相册创意、时间线叙事、BGM 控制。
- 调整点：固定 1280 viewport 不利于移动端；部分偏个人化资料建议降级为待核验。

## 4. fengmingjiongjun_site
- 定位：轻量核验型、稳定展示型。
- 优点：主题切换、曲目搜索、隐私/核验原则、低依赖、结构简洁。
- 适合保留：明暗主题、搜索、隐私保护说明、低依赖实现。

## 整合版架构
1. Header：Logo、锚点导航、社交快捷入口、BGM、主题切换、移动菜单。
2. Hero：主视觉、口号、核心标签、统计卡、CTA。
3. Profile：基础档案、风格关键词、核验说明。
4. Music：分类 Tabs、搜索、歌曲卡片、iframe 播放器、平台源切换。
5. Platforms：平台卡片墙、商务联系方式。
6. Timeline：错位时间线，移动端单列。
7. Gallery：本地图像、瀑布流、卷轴相册、灯箱。
8. Sources：资料来源、版权说明、整合原则、文件结构。
9. Footer：非官方声明、快速链接、返回顶部。
