## 4. 照片与视觉素材（已本地化）

本版已按需求把页面中展示的照片/视觉素材下载到项目内，不再把图片 `src` 指向站外外链。

本地文件位置：

```text
assets/images/photos/          原图或较高清文件
assets/images/photos/thumbs/   页面照片墙与 Hero 使用的缩略图
assets/data/photos.json        图片标题、来源核验链接、本地路径
```

已本地化的来源类型：

| 来源类型 | 用途 | 本地化说明 |
|---|---|---|
| Bilibili 收藏集图片页 | 风象双子、旗袍囧菌、人鱼囧菌等视觉图 | 下载到 `assets/images/photos/`，并生成缩略图 |
| 360百科图册 | 补充肖像素材 | 下载到本地，保留图册页核验链接 |
| 堆糖公开图集/相册 | 照片墙补充素材 | 第三方来源，已在 JSON 中标注来源与核验备注 |

页面实现说明：

- 照片墙 `<img>` 使用 `photo.src`，均为 `assets/images/photos/thumbs/...` 本地路径。
- “本地大图”按钮使用 `photo.hires`，均为 `assets/images/photos/...` 本地路径。
- “来源核验”按钮仍会打开外部页面，但只是用于查看出处，不参与图片加载。
- 旧版使用的新浪/微博图片外链可能触发 403 或防盗链，不再作为页面图片依赖。
- 非官方第三方图集仅作为补充视觉素材；公开发布或商用前应确认授权。
