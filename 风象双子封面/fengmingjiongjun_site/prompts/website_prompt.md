# 继续优化网页的提示词

你是一名高级前端设计师和资料核验助手。请基于 `data/facts.json` 中已经核验的资料，为歌手“封茗囧菌 / Mandi Sa”制作一个精美、响应式、粉白淡黄浅蓝配色的个人音乐主页。

要求：
1. 不要编造未经核验的个人隐私信息；身高、体重、民族、住址等不放入主页面。
2. 页面必须包含：Hero、个人资料、代表曲目、平台主页、照片/封面视觉、音乐时间线、资料来源。
3. 热门曲目只称为“热门 / 代表曲目”，不要写成永久固定排名。
4. 平台直链只使用 `facts.json` 中已给出的 URL；未核验平台只写“同ID站内搜索”。
5. 图片采用外链显示，并在无法加载时使用本地占位图。
6. 视觉风格：玻璃拟态、柔和渐变、圆角卡片、优雅音乐人主页、适配桌面和平板手机。
7. 输出纯 HTML/CSS/JS，不依赖外部库。

可选增强：
- 添加歌曲搜索、明暗主题切换、时间线动效。
- 添加“资料核验说明”与“不是官方网页”的标识。
